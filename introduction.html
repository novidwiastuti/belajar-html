Introduction

HTML

HTML (Hyper Text Mark Up Language) merupakan semacam format struktur dasar dari halaman web, yaitu kumpulan kode yang berisi informasi halaman web.
HTML dibaca oleh web browser untuk ditampilkan visualisasinya.


CSS

CSS (Cascading Style Sheet) digunakan untuk mengatur gaya tampilan halaman web.


Java Script

Java Script digunakan untuk interaksi halaman web.
